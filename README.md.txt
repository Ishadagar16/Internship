*****  Project Overview  *****
This is a beginner Python project that collects user information and displays a friendly welcome message.

#  Objectives
- Learn Python basics
- Understand variables and input/output
- Practice f-strings formatting

#  Setup Instructions
1. Install Python
2. Run: python personal_intro.py

# Project Structure
- personal_intro.py  (Main program)
- README.md  (Documentation)
- requirements.txt  (Dependencies)
- screenshot.png  (Output proof)

# What I Learned
- How to use input()
- How to store data in variables
- How to print formatted output
- Basic project structure
